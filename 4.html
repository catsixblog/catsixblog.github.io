用手机写blog.喵

发布 : 2019-01-21 分类 : 教程分享 浏览 :

先下载一个终端模拟器

  1. 21世纪的终端-NeoTerm 2.0.3-cc862be

等待下载好进入

  • apt install git 安装git

设置git

git config –global user.name “你的github用户名”
git config –global user.email “注册github的邮箱”

  • apt install nodejs 安装nodejs
  • npm install hexo-cli -g 安装hexo

去github创建一个储存库 可以参考 这个文章

创建本地运行环境

mkdir blog
cd blog
hexo init
npm install

编辑“_config.yml”配置文件

pkg install nano
nano _config.yml


在文件最下面“deploy”按照图里设置
击底部快捷栏“CTRL”(变深色)，然后输入“o”保存，再按回车键保存点击底部快捷栏“CTRL”，按“x”退出编辑

最后一步

npm install hexo-deployer-git –save 安装部署服务

部署到github

hexo clean 清除缓存
hexo generate
hexo deploy 部署的时猴要输帐号密码
